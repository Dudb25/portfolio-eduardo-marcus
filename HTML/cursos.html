CURSOS

SQL para Análise de Dados
Desenvolvi habilidades em SQL voltadas para análise de dados, incluindo consultas complexas, uso de joins, agregações e manipulação de grandes volumes de informação. Tenho capacidade de extrair insights relevantes e organizar dados de forma eficiente para apoiar a tomada de decisões.
Ciência de Dados com Python
Adquiri fundamentos em ciência de dados utilizando Python, com foco em análise exploratória, manipulação de dados e uso de bibliotecas como Pandas e NumPy. Desenvolvi a capacidade de interpretar dados e transformá-los em informações úteis.
Power BI para Data Analysis
Aprendi a criar dashboards interativos e relatórios visuais no Power BI, conectando diferentes fontes de dados e aplicando transformações. Tenho habilidade em apresentar informações de forma clara e estratégica, facilitando análises e decisões.
Project Management Essentials
Desenvolvi conhecimentos em gestão de projetos, incluindo planejamento, definição de escopo, acompanhamento de tarefas e controle de prazos. Tenho noção de organização de equipes e foco na entrega de resultados alinhados aos objetivos.
Scrum Foundations
Aprendi os princípios e práticas do Scrum, incluindo papéis, cerimônias e artefatos. Tenho entendimento de como aplicar metodologias ágeis para melhorar a produtividade, colaboração e adaptação em projetos.
